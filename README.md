# Wildlife Donation Website with Razorpay Integration

Welcome to the Wildlife Donation Website! This website allows users to make donations for wildlife conservation efforts using the Razorpay payment gateway interface.

## Technologies Used

- HTML
- CSS
- Razorpay Payment Gateway Integration

## Features

- Home Page: Provides information about wildlife conservation efforts and the importance of donations.
- Donation Page: Allows users to input their donation amount and make a payment using Razorpay.
- Thank You Page: Displays a thank you message after a successful donation.

## Installation

1. Clone the repository:

    ```
    git clone https://github.com/your-username/wildlife-donation-website.git
    ```

2. Open the project directory in your code editor.

3. Replace placeholders in the code with your Razorpay API keys.

## Usage

1. Navigate to the project directory and open `index.html` in your web browser.

2. Explore the website, learn about wildlife conservation efforts, and proceed to the donation page.

3. Enter the desired donation amount and complete the payment process using Razorpay.

4. After successful payment, you will be redirected to the Thank You page.

## Razorpay Integration

This website utilizes the Razorpay payment gateway for secure online transactions. Make sure to replace the placeholder API keys with your actual Razorpay API keys for integration to work properly.
